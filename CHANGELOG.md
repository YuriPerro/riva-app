# Changelog

All notable changes to Riva will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.5] - 2026-03-23

### Fixed
- Queued pipelines now display correctly instead of showing as cancelled

## [0.1.4] - 2026-03-23

### Added
- Sprint navigation to dashboard

## [0.1.3] - 2026-03-23

### Added
- Initial public release
- Dashboard with sprint overview and pipeline status at a glance
- Work items management (Tasks, Bugs, PBIs, Features, Epics) with filtering and search
- Pipeline monitoring with live status indicators and progress bars
- Pull request review — approve, reject, filter by status and repository
- Release pipeline tracking with environment deployment approval
- AI Standup Summary via OpenAI with configurable lookback period
- Native OS notifications for PR reviews, pipeline failures, and work item mentions
- 9 dark themes: Midnight, Phosphor, Dracula, Monochrome, Crimson, Catppuccin, Sunburn, Claude, Red Dead
- Cross-platform builds — macOS, Windows, Linux
